# Portfólio Pessoal

## Sobre o projeto

Este projeto foi desenvolvido como atividade da disciplina de Desenvolvimento Front-End.

O objetivo é apresentar um portfólio profissional contendo informações pessoais, habilidades, projetos e formas de contato.

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript


## Estrutura do projeto

- index.html
- sobre.html
- projetos.html
- contato.html

## Funcionalidades

- Navegação entre páginas
- Layout responsivo
- Animações com JavaScript
- Formulário de contato
- Integração com EmailJS
